# Hyprland Keybindings Reference

## Basic Applications
- `SUPER + Q` - Launch terminal (Kitty)
- `SUPER + E` - File manager (Thunar)
- `SUPER + R` - Application launcher (Rofi)
- `SUPER + C` - Close active window

## Window Management
- `SUPER + F` - Toggle fullscreen
- `SUPER + SHIFT + V` - Toggle floating
- `SUPER + T` - Toggle split (dwindle)
- `SUPER + P` - Toggle pseudo (dwindle)

## Focus Navigation (Vim Keys)
- `SUPER + H` - Focus left
- `SUPER + J` - Focus down
- `SUPER + K` - Focus up
- `SUPER + L` - Focus right

## Focus Navigation (Arrow Keys)
- `SUPER + Left/Right/Up/Down` - Focus in direction

## Move Windows (Vim Keys)
- `SUPER + SHIFT + H` - Move window left
- `SUPER + SHIFT + J` - Move window down
- `SUPER + SHIFT + K` - Move window up
- `SUPER + SHIFT + L` - Move window right

## Resize Windows (Vim Keys)
- `SUPER + CTRL + H` - Resize left (-50px)
- `SUPER + CTRL + J` - Resize down (+50px)
- `SUPER + CTRL + K` - Resize up (-50px)
- `SUPER + CTRL + L` - Resize right (+50px)

## Resize Windows (Arrow Keys)
- `SUPER + CTRL + Left/Right/Up/Down` - Resize in direction

## Workspaces
- `SUPER + 1-9/0` - Switch to workspace 1-10
- `SUPER + SHIFT + 1-9/0` - Move window to workspace 1-10
- `SUPER + S` - Toggle scratchpad
- `SUPER + SHIFT + S` - Move to scratchpad
- `SUPER + Mouse_Scroll` - Cycle workspaces

## Clipboard Manager (Cliphist)
- `SUPER + V` - Open clipboard history (Wofi)
- `SUPER + Period` - Open clipboard history (Rofi)

## System Controls
- `SUPER + M` - Exit Hyprland
- `SUPER + SHIFT + E` - Logout menu (wlogout)

## Screenshots
- `Print` - Screenshot selection (with Satty editor)
- `SHIFT + Print` - Screenshot full screen (with Satty editor)

## Utilities
- `SUPER + SHIFT + C` - Color picker (Hyprpicker)

## Media Keys
- `XF86AudioRaiseVolume` - Volume up 5%
- `XF86AudioLowerVolume` - Volume down 5%
- `XF86AudioMute` - Toggle mute
- `XF86AudioMicMute` - Toggle mic mute
- `XF86MonBrightnessUp` - Brightness up 5%
- `XF86MonBrightnessDown` - Brightness down 5%
- `XF86AudioNext/Prev/Play/Pause` - Media controls (playerctl)

## Mouse
- `SUPER + LMB` (drag) - Move window
- `SUPER + RMB` (drag) - Resize window

---

**Tip**: Most keybindings follow vim-style navigation (h/j/k/l) for consistency!
